# FdF
3D Wireframe Viewer
<br>
This project is about creating a simplified 3D graphic representation of a given relief landscape.

### Examples

![mars.fdf map](./other/mars.png)
![t1.fdf map](./other/t1.png)

## Usage:

Go to `fdf` directory and:
```sh
$ make
$ ./fdf mars.fdf
```
